
# Tecmagest

Este é o repositório do projecto Tecmagest, um sistema web inovador desenvolvido por angolanos, focado em soluções de contabilidade e gestão para empresas.

## Principais Funcionalidades
- Gerenciamento de serviços e planos
- Interface moderna e responsiva
- Componentização em React
- Estrutura organizada para fácil manutenção

## Estrutura do Projeto
- `App.jsx`, `index.js`: Arquivos principais da aplicação React
- `components/`: Componentes reutilizáveis organizados por funcionalidade
- `public/`: Arquivos públicos e estáticos (incluindo `index.html` e fontes)
- `images/`: Imagens utilizadas no site

## Como rodar o projeto
1. Instale as dependências:
	```bash
	npm install
	```
2. Inicie o servidor de desenvolvimento:
	```bash
	npm start
	```
